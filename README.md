# NX4Board - 智慧型 OBD2 行車儀表與偵測系統
![演示動畫](rec.gif)

NX4Board 是一款專為現代 (Hyundai) 與起亞 (Kia) 車系開發的智慧行車儀表應用。透過藍牙 OBD2 連接，即時讀取並顯示車輛深度資訊，同時整合政府公開資料，提供精準的測速照相與路段速限偵測。

## 🌟 核心特色

- **即時診斷儀表**：動態讀取 OBD2 資料，提供美觀且流暢的行車數據顯示介面。
- **深度車輛支援**：
  - 支援現代 (Hyundai) / 起亞 (Kia) 全系油車與油電 (Hybrid) 車款。
  - **通用資料**：時速、引擎轉速 (RPM)、水溫、經緯度。
  - **進階資料**：四輪獨立胎壓 (TPMS)、總里程 (ODO)、油箱容量、渦輪壓力 (Turbo)、油電車高壓大電池電量 (SOC)、倒車檔位 等。
- **智慧安全導航**：
  - **固定式測速偵測**：使用政府公開資料，具備儀表提示與語音告警 (TTS)。
  - **路段速限偵測**：使用政府公開資料偵測附近道路速限標誌，即時更新面板速限指示。
- **數據同步功能 (Web Socket)**：
  - 支援將讀取到的車輛資料組成 JSON，透過 Web Socket 後送至其他設備或伺服器。

## 📡 數據後送格式 (JSON 範例)

本 App 會將車輛狀態打包為標準 JSON 格式傳送：

```json
{
  "_type": "location",
  "tid": "obd",
  "fuel": 50.5,
  "mileage": 26500,
  "avg_fuel": 15.2,
  "tires": {
    "fl": 33,
    "fr": 33,
    "rl": 33,
    "rr": 33
  }
}
```

## 🛠️ 下載與部署

### 快速安裝
1. 下載專案根目錄下的 `app-release.apk`。
2. 在您的 Android 設備上進行安裝（建議使用橫向大螢幕設備以獲得最佳體驗）。
3. 使用藍牙配對您的 ELM327 OBD2 接收器。

### 自動化編譯 (開發者)
本專案內含 `deploy.sh` 腳本，可自動處理版本號遞增與編譯：
```bash
./deploy.sh
```

## 📋 系統需求
- **硬體**：Android 設備 (建議 Android 9.0+)，藍牙 ELM327 OBD2 接收器。
- **軟體**：Flutter 3.0+ (若需自行編譯)。
